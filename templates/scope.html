Future Scope

As this is unsupervised learning, choosing the clustering model will be
a big task which will be most efficient. It will involve trying out
different models, tuning the hyperparameters and select the correct model to go.

Few difficulties or obstacles such as one coding error can give rise
to multiple errors and therefore clustering the errors into groups
may not give information about this relations between different errors
which will make it difficult to backtrack and trace the actual problem.
This will be a part to work on in the later stage.

If error clustering can be done efficiently or the error groups can be identified,
then we can work towards building a suggestion model to these separate error
groups which just guide how to debug a particular kind of errors.
